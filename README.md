# 📚 RSET Question Paper Finder

A Firefox extension that helps students quickly search and filter question papers by subject and semester from the RSET exam portal.

---

## 🚀 Features

- 🔍 Search papers by subject
- 🎯 Filter by semester
- ⚡ Fast extraction from website
- 🧠 Smart matching (partial words supported)

---

## 🧠 Problem Solved

Students struggle to find question papers because they are scattered and not searchable.  
This extension makes it instant and easy.

---

## 🛠️ Tech Used

- JavaScript
- Firefox Extension API
- DOM Manipulation

---

## ▶️ How to Use

1. Open Firefox  
2. Go to `about:debugging`  
3. Click **This Firefox**  
4. Click **Load Temporary Add-on**  
5. Select `manifest.json`  
6. Open the RSET question paper page  
7. Use the extension
